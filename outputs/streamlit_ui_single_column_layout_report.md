## Layout change

The main page was changed from a two-column layout:

- left column: detected entities + Top-k + method comparison
- right column: result details + export

to a single vertical flow:

1. `1. Query input`
2. `2. Detected query entities`
3. `3. Top-k results`
4. `4. Result details`
5. `5. Method comparison`
6. `6. Export results`

`Result details`, `KG subgraph`, `Mini KG graph`, source evidence, and evidence tables now use the full main content width instead of being constrained in the right column.
